Danh sách các thẻ HTML

Cấu trúc tài liệu

        <!DOCTYPE html>
        <html>
        <head>
            <title>Tiêu đề trang</title>
        </head>
        <body>
            Nội dung trang web
        </body>
        </html>

Thẻ meta

        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <meta name="description" content="Trang web của tôi">

Tiêu đề và văn bản

        <h1>Tiêu đề chính</h1>
        <p>Đây là một đoạn văn bản.</p>
        <br> (Xuống dòng)
        <hr> (Kẻ đường ngang)

Danh sách

        <ul>
            <li>Mục 1</li>
            <li>Mục 2</li>
        </ul>
        
        <ol>
            <li>Mục 1</li>
            <li>Mục 2</li>
        </ol>

Liên kết

        <a href="https://example.com" target="_blank">Truy cập trang web</a>

Hình ảnh và đa phương tiện

        <img src="image.jpg" alt="Mô tả hình ảnh">
        <audio controls>
            <source src="audio.mp3" type="audio/mpeg">
        </audio>
        <video width="320" height="240" controls>
            <source src="video.mp4" type="video/mp4">
        </video>

Bảng

        <table border="1">
            <tr>
                <th>Cột 1</th>
                <th>Cột 2</th>
            </tr>
            <tr>
                <td>Dữ liệu 1</td>
                <td>Dữ liệu 2</td>
            </tr>
        </table>

Biểu mẫu

        <form>
            <label for="name">Tên:</label>
            <input type="text" id="name" name="name">
            <button type="submit">Gửi</button>
        </form>

Thẻ định dạng

        <strong>In đậm</strong>
        <em>In nghiêng</em>
        <mark>Bôi vàng</mark>

Thẻ script

        <script>
            alert("Xin chào!");
        </script>

Thẻ kiểu dáng

        <style>
            body { background-color: lightblue; }
        </style>

Thẻ điều hướng

        <header>Phần đầu trang</header>
        <nav>
            <a href="#">Trang chủ</a>
            <a href="#">Giới thiệu</a>
        </nav>

Thẻ nhúng

        <iframe src="https://example.com" width="600" height="400"></iframe>

Thẻ khác

        <details>
            <summary>Chi tiết</summary>
            Nội dung mở rộng.
        </details>
